================================================================
1) Initial Hypothesis and Approach
================================================================
Hypothesis:
- If a shutdown/kill-switch password exists in a simple ELF binary, it is commonly checked in:
  - main()
  - argument parsing logic
  - string compare branches (strcmp/strncmp/memcmp)
  - hardcoded string literals in .rodata

Approach:
- Enumerate imports to detect compare/string routines.
- Enumerate functions to find likely control logic entry points.
- Decompile main first (most probable location in small binaries).
- Confirm exact literal being compared and output behavior.

Reasoning:
- Small binaries often perform command-line password checks directly in main.
- Presence of strcmp import is a strong indicator of literal string comparison.
